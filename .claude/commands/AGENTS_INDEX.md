# AIOS Agents Index

## Tous os Agents Disponíveis

Use qualquer um dos comandos abaixo para invocar o agent correspondente:

### Core Agents

#### `/aios-master`
**AIOS Master - Orchestrator and Developer Coordinator**
- Orquestra todos os outros agents
- Visão geral e estratégia do projeto
- Coordenação de fluxos multi-agent

#### `/dev`
**Developer Agent - Software Development Specialist**
- Desenvolvimento full-stack
- Correção de bugs
- Implementação de features
- Code reviews e refactoring

#### `/architect`
**Architect Agent - System Design Specialist**
- Design e arquitetura de sistemas
- Seleção de technology stack
- Planejamento de escalabilidade

## Como Usar

Simplesmente digite o comando do agent desejado no Claude Code:

```
/dev
```

O agent será carregado com sua persona completa e instruções de operação específicas.

## AIOS CLI Commands

Para comandos AIOS (init, install, update, doctor, etc.), use:

- `/aios` - Comando genérico
- `/aios-init` - Criar novo projeto
- `/aios-install` - Instalar/configurar AIOS
- `/aios-update` - Atualizar AIOS
- `/aios-doctor` - Diagnóstico do sistema
- `/aios-info` - Informações do sistema
- `/aios-version` - Versão instalada
