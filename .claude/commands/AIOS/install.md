# AIOS Install Command

Execute: `aios-core install $ARGUMENTS`

This command sets up AIOS in an existing project directory.

Usage:
- `/aios-install` - Install AIOS with default settings
- `/aios-install --template default` - Install with default template
- `/aios-install --template minimal` - Install with minimal template
- `/aios-install --template enterprise` - Install with enterprise template
- `/aios-install --quiet` - Install silently without verbose output
- `/aios-install --dry-run` - Simulate installation without making changes
