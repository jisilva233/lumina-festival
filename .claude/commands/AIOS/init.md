# AIOS Init Command

Execute: `aios-core init $ARGUMENTS`

This command creates a new AIOS project with an interactive setup wizard.

Usage:
- `/aios-init my-project` - Create a new project named "my-project"
- `/aios-init my-project --template minimal` - Create with minimal template
